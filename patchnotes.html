Season 1
From Beta
May 29, 2026

⚙ Generator Engine (Complete Rewrite)

  • ▸ Generator spawning, mutations, drops, selling, and upgrades now handled by the native GCore Java plugin for vastly improved performance.
  • ▸ New /sell command replaces the old sell-block system. Sells all gen drops in your inventory with a clean receipt showing multiplier, XP, and progress.
  • ▸ Offline Earnings Engine added. Players with 10+ placed gens earn 50% of their online income for up to 24 hours while offline.
  • ▸ 60 generator tiers (30 money + 30 token forges) with smooth level-up token scaling.

🌱 Farming Overhaul

  • ▸ 68 walk-through crops with MMO-style progression. Flowers, mushrooms, fungi, and more scale from Wheat at Prestige 0 to endgame crops at Prestige 500+.
  • ▸ New Crop Leaderboard with massive hologram display.
  • ▸ Farmer's Lunchbox now grants +1 Hoe XP on use.
  • ▸ Harvest animations replaced with a lag-free system.

🐾 Pets & Eggs

  • ▸ All eggs now use vanilla dragon eggs with correct per-tier colors and incubation time lore. No more all-pink eggs.
  • ▸ Pet abilities rebalanced across the board (new key types, gen drops, and essence rewards for multiple pets).
  • ▸ Sneak-to-pickup safety lock added to prevent accidental egg collection.

💰 Economy & Shops

  • ▸ In-Game Credit Shop added (/buy). Full virtual currency system with dynamic sales engine, rainbow border animation, and Discord purchase logging.
  • ▸ The Nexus shop hub redesigned with live wallet display and cleaner tab navigation.
  • ▸ Token Enchant Shop rebalanced. Gen slot pricing is now exponential. Multiplier values nerfed to +0.01/+0.02/+0.05x with daily cap.
  • ▸ Grand Market rebuilt with flat item lists and top-tab navigation (no more nested folders).
  • ▸ Banknote system added. Withdraw money into tradeable paper items with /banknote.

👑 Prestige & Rebirth

  • ▸ Prestige GUI split into Main + Shop. Level requirement is now linear (+5 per prestige) instead of escalating tiers.
  • ▸ Prestige rewards rebalanced: Relics replaced with Genslots and Prestige Points. Streak, pity, and milestone info displayed.
  • ▸ Rebirth GUI split into Main + Shop. Farm Yield and Mine Fortune skills removed.

🔑 Crate Keys & Loot Boxes

  • ▸ Crate system completely rewritten with actionbar roll animations, strict rewards, and multi-open support.
  • ▸ Keys now use proper type mapping. Keys from weather events and admin abuse stack correctly with keys from crates and shops.
  • ▸ All loot boxes rebalanced for the stabilized economy. Eco, multiplier, and genslot ranges tightened across Core Box, Premium Core, Mini Box, and Maxi Box.

⚔ PvP & Combat

  • ▸ Mace weapon class added to the Armory with Density enchants, purchasable with Essence across all armory tiers.
  • ▸ Paintball game mode and gun system removed (to be reworked in a future update).

🌞 Weather & Admin Abuse

  • ▸ Weather panel simplified to single-page Cataclysms only. Ambient weather automation removed.
  • ▸ Weather egg drops now display correct per-type colors and incubation lore.

⛏ Private Mines

  • ▸ All ore prices rebalanced for the new economy (Coal Ore: 10 → 5, Diamond Ore: 250 → 100, Netherite: 5,000 → 1,500).
  • ▸ Rain enchant block prices added (Iron/Gold/Diamond/Emerald blocks).
  • ▸ Forger Dan shop now features progressive unlock (items lock behind Prestige/Level/Mine Tier).

🏆 Ranks & Daily/Monthly Rewards

  • ▸ All ranks now display premium hex gradient names and emoji chat tags.
  • ▸ Generator rewards synced to the new tier config (e.g. "Silica Gen" instead of "Legendary Stone").

🗪 Chat & Social

  • ▸ Math Reaction Games added to chat. Random math problems appear; first correct answer wins 5K-15K tokens + a chance at Credits.
  • ▸ All gameplay tags upgraded to premium hex gradients (Farmer, Fisher, Miner, Collector).

🎮 Events & Minigames

  • ▸ The Great Pollen Event added: A seasonal bee event with AI bees, floor mechanics, and a dedicated leaderboard.
  • ▸ Parkour engine made fully client-sided. Multiple players can now run parkour simultaneously with per-player fake blocks.
  • ▸ Easter Egg Hunt now gives +1 Gen Slot per egg found. Entity-based click detection replaces the old block-based system.

🛠 Quality of Life

  • ▸ New player starter kit rebalanced: 7 Tier 1 gens + 1 Common Egg.
  • ▸ Leaderboard expanded from 6 to 10 pages. Page 1 changed to Richest Players (Vault balance).
  • ▸ Reputation cost reduced from $10M to $100K per point.
  • ▸ /help tutorial rewards changed from Relics to escalating key types (farmer → genesis).
  • ▸ Fishing system now grants XP from selling fish. Ice pond progression steeper (1.5x → 1.8x scaling).
  • ▸ Nuclear Reset admin commands added for complete player or server-wide wipes.

🔧 Bug Fixes

  • ▸ Fixed weather-dropped eggs all appearing pink instead of their correct rarity color.
  • ▸ Fixed crate keys from weather events and admin abuse not stacking with keys from other sources.
  • ▸ Fixed note condenser auto-condensing inside custom GUIs (now only triggers in chests, hoppers, barrels, droppers, dispensers).
  • ▸ Fixed egg hunt stats hologram overlapping or disappearing on rejoin.
  • ▸ Fixed parkour blocks being shared between players causing visual conflicts.
